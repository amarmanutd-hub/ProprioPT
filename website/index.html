You write the plan. They use a code.

Same idea as MedBridge-style HEP invites — without making the patient build an account first.

  1. 01

    PT builds the session

    Pick exercises, reps, and dosing. Proprio generates an access code (and optional link) for that patient.

  2. 02

    Patient enters the code

    On their phone or laptop — camera on, short setup, then the moves you assigned. Squats get live form cues; other moves are counted clearly.

  3. 03

    Summary stays with them

    Reps, status, and form notes show on screen. Video never uploads. They can save a copy to share back if you want it.
